The parameters should be 1,10,100

1: the same build number between upstream and downstream job,if the build number is 34 in upstream job then the build number will be 34

10: 10 times build number,for example,if the build number is 34 in upstream job then the build number will be 340

100:100 times build number,for example,if the build number is 34 in upstream job then the build number will be 3400
