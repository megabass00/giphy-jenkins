This column allows you to more flexibly configure what types of statuses to show. Options include

  • Only show the last status - Choose this if all you ever want to show is the status of the last build.
  • Builds to show - For example, if you want to show multiple statuses, but once a build is a month old, you want to stop showing it. Choose "0" to show all statuses.
  • Time display options
    • Days/hours ago - for example "12 days" or "2 hours"
    • Date/Time - Prefer Date - will show the date (e.g. "5/25/2016") most of the time, but will show the time if the build was today.
    • Date/Time - Prefer Date/Time - the same as the previous option, except when there is only one status in the column, the date/time is shown.
  • Colorblind options - Color-coding can cause accessibility concerns. Choose a colorblind hints option if this will be useful to you.
