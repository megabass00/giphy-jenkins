Das lokale Verzeichnis wird benötigt, um Artefakte und Metadaten herunterzuladen. Dieses Verzeichnis kann gelegentlich gelöscht werden, wodurch ggfs. bereits heruntergeladene Artefakte neu geladen werden müssen. Der Pfad kann auf ein existierendes lokales Maven Verzeichnis zeigen oder ein separates Verzeichnis sein. Definieren Sie hier das Verzeichnis gegen das verbunden wird:

  • Verzeichnis Id: eine eindeutige id für dieses Verzeichnis
  • Verzeichnisstruktur: Die Verzeichnisstruktur, üblicherweise 'default'
  • Verzeichnismanager: ausgew&aauml;hlt wenn auf dem Server eine Verzeichnismanager verwendet wird, nicht ausgewählt wenn nur statische Inhalte ausgeliefert werden.
  • Url: Die URL des Verzeichnisservers
  • Benutzer: Der Benutzer, der für die Verbindung verwendet werden soll.
  • Passwort: Das Passwort des Benutzers

Benutzer und Passwort können auf Job Ebene zu überschrieben werden. Note: Wegen dieses Fehlers https://issues.sonatype.org/browse/NEXUS-3776 sollte mindestens Nexus 1.8.* oder höher eingesetzt werden.
